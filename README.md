🔍 Pokédex Project<br>
Ein moderner, interaktiver Pokédex, der Daten direkt aus der PokéAPI bezieht. Dieses Projekt wurde entwickelt, um Pokémon-Daten übersichtlich darzustellen und Funktionen wie Suche, Filterung und Detailansichten zu bieten.<br><br>

🚀 Features<br>
Echtzeit-Daten: Dynamisches Laden von Pokémon-Informationen über die REST-API.<br>

Suche & Filter: Finde deine Lieblings-Pokémon blitzschnell nach Namen oder ID.<br>

Detailansicht: Klicke auf ein Pokémon, um detaillierte Statistiken (Base Stats), Typen, Fähigkeiten und mehr zu sehen.<br>

Responsive Design: Optimiert für Desktop, Tablets und Smartphones.<br>

Infinite Scrolling / Pagination: Flüssiges Durchstöbern der gesamten Pokémon-Liste.<br><br>

🛠️ Technologien<br>
Dieses Projekt nutzt einen modernen Tech-Stack für optimale Performance und Wartbarkeit:<br>

Frontend: HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+)<br>

API: PokéAPI – Die RESTful Pokémon API<br>

Icons & Fonts: FontAwesome / Google Fonts (falls verwendet)<br><br>

📦 Installation & Setup<br>
Um das Projekt lokal auszuführen, folge diesen einfachen Schritten:<br>

Repository klonen:<br><br>

Bash<br>
git clone https://github.com/maximilianbese/Pokedex.git<br>
In das Verzeichnis wechseln:<br><br>

Bash<br>
cd Pokedex<br><br>
Projekt starten:<br>
Öffne die index.html direkt in deinem Browser oder nutze eine Extension wie "Live Server" in VS Code.<br><br>

📖 Nutzung<br>
Nach dem Starten der Anwendung siehst du eine Liste von Pokémon.<br>

Nutze die Suchleiste, um nach einem spezifischen Namen zu filtern.<br>

Scrolle nach unten, um weitere Pokémon nachzuladen.<br>

Ein Klick auf eine Pokémon-Karte öffnet ein Modal oder eine Detailseite mit zusätzlichen Informationen wie Gewicht, Größe und den Statuswerten (Angriff, Verteidigung, etc.).<br><br>
